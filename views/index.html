API Project: File Metadata Microservice

Requirements

  • I can submit a form object that includes a file upload. (POST: /api/fileanalyse/single)
  • I can submit a form object that includes multiple (up to 5) file uploads. (POST: /api/fileanalyse/multiple)
  • The from file input field has the "name" attribute set to "upfile". We rely on this in testing.
  • When I submit something, I will receive the file name, file type and size in bytes within the JSON response.
